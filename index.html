    MANUAL설명서
    재화

    · 크레딧: 기본 화폐. 상점 구매·코드 강화에 사용.

    · TOKEN: EVENT 기능 재화. OPS SHOP에서 사용.

    · COIN: 희귀 시즌 재화. SEASON SHOP에서 사용.

    · OneDay: ZERO-DAY 성장 재화. PVP > PVE 순으로 획득.

    · 취약점: ZERO-DAY 입장권. 스캔 1% 확률 / 조각 50개 제작.

    모드

    · NORMAL: 기본 해킹. 패널티 없음.

    · RISK: 성공률 -15%p, 보상 ×2. 실패 시 에너지 +1 소모.

    · EXTREME: Lv.5+, 성공률 -35%p, 보상 ×3. 실패 시 에너지 +2 소모.

    서버 루트

    · 외부 루트: 레벨 요구 -1, 보안 ×0.85, 보상 ×0.8.

    · 내부 루트: 기본 설정.

    · 코어 루트: 레벨 요구 +2, 보안 ×1.2, 보상 ×1.35.

    데이터 타워

    · 100층, 10챕터 × 10층 구조. 에너지 1 소모.

    · BREACH / SHIELD / FOCUS / EXIT 4액션 턴제 전투.

    · 첫 클리어 보상 > 반복 보상(80% 감소). 챕터 클리어 보상 별도.

    EVENT

    · WEEKLY CHALLENGE: 매주 월 05:00 KST 리셋. SCORE·TOKEN 획득.

    · PASS: 월간 시즌제. 패스 포인트 100 = 티어 1 (최대 30티어).

    · 시즌 1: 2026-05-01 시작, 테마 ZERO-DAY.

    ZERO-DAY

    · 온보딩 완료 → 취약점 1개 지급, PVE·PVP 해금.

    · PVE 입문 난이도: 하루 1회 무료.

    · PVP: 로그인 필요. 비동기 스냅샷 기반 매칭.

    · 명령어: scan/probe/enum(정찰), breach/inject/elevate/bypass(침투), mask/cloak/wipe/scrub(은폐), spoof/reroute/jam/decoy(교란), exfil/exit/disconnect(탈출).
